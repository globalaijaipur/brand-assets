# Global AI Jaipur Creative Assets

This repository contains open creative assets and branding resources for **Global AI Jaipur** and the **Global AI Community**.

These resources are provided to support organizers, volunteers, partners, and community members in creating consistent visuals for events, meetups, workshops, and announcements.

## Contents

- Logos and brand assets
- Community banners
- Brand guidelines

## Usage

You are free to use these assets for **Global AI Community and Global AI Jaipur related activities**, including meetups, workshops, and educational initiatives.

Please avoid modifying official logos in ways that alter brand identity.
